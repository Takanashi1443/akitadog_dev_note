# ArrowLine

## 参照するアセット

- ArrowLine
- RopeLine(ほぼ同じ方法で作られている)

## 各ノードの意味

#### Split

UVを分割する。

RGBAという名前になっているが特にRGBAである必要はない。

1つめの値であるUを1つめのout(R)に、2つめの値であるVを2つめのout(G)に分けている。

この時点で、上側には左が0、右が1のグラデーション、下側には上が0、下が1のグラデーションが出来ている。

#### Multiply、Fraction(上側)

MultiplyでTiling（整数値）倍し、Fractionで1以下のあまりを使用することで
左が0、右が1の繰り返しパターンをTilingの数だけ増やす。

Fractionはここで入れなくても結果的に同じになる。

#### One Minus(上側)

繰り返しパターンを、左が1、右が1になるようにする。

#### TimeをAdd(上側)

1秒に1増加する値を加える。

この後、最後にFractionすることで1秒ごとに0~1の値が漸増するようになっている。

なお、Sine Time、Cosine Timeは-1~1で周期的に変化する時間を与えるものだが、
周期が固定なので通常はTimeとSineノードを組み合わせた方が良さそう。

#### SubtractとAbsolute(下側)

0.5を引くことで上が-0.5、中間が0、下が0.5のグラデーションにした上で、絶対値を取って上下0.5にしている。

#### Subtract

上（0~1＋Timeの値）から下（0~0.5）を引くと斜めのグラデーションになる。

下は0~0.5の範囲で良いのか？　という気がするが、これは傾きを決めているだけで結果的には問題ない。

#### Fraction、Step、Lerp

小数点以下を取ったのちに、Step以上の値を1、未満は0にすることで2値化する。

Lerpは線形補間であり、入力A×(1-T)＋入力B×Tを行う。ここではTが0か1なので、Tが1ならAに、0ならBに等しくなる。

StepとLerpの組み合わせはこういう幾何学的テクスチャを作るのにかなり便利そうである。
