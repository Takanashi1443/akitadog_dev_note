# Unityルール

## 基本関係

- ShibaShelfのコンポーネントの名前空間はShibaShelf.(アセット名)とする。

## プロジェクトを作った時

### 改行コードをLFにする

.editorconfigファイルをUnityプロジェクト直下に配置する。

既存のプロジェクトからコピーしてくる。

参考ページ:

- [Visual Studioで使う改行コードを指定する方法](https://h-sao.com/blog/2019/07/26/how-to-keep-line-feed-code-by-visualstudio-2019/)

## ファイル・フォルダ構成・ファイル名関連

- Assets直下に「今作っているゲーム」の名前のフォルダを作り、その中で完結させる。
- Editorフォルダはそのコンポーネントが入っているフォルダ内に作る。（階層を分岐させない）
